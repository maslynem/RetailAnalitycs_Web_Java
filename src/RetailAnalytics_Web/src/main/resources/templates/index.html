Информация о проекте

Проект выполнен пиром maslynem

Web-приложение разработано на языке Java версии 17

Web-приложение реализовано с использованием MVC-фреймворка (Spring)

Рендеринг страниц осуществляется на стороне сервера (использование технологии Server-Side Rendering)

Имеется навигационное меню, обеспечивающее переход к основным разделам приложения: «Данные» и «Операции»

Раздел «Данные» содержит подразделы, который позволяет совершать CRUD-операции по всем таблицам

Раздел «Операции» содержит:

  • Блок с возможностью самостоятельного ввода пользователем SQL-запроса для работы с данными в базе
  • Блок, содержащий все возможные для вызова функций и процедур, хранящихся в БД
